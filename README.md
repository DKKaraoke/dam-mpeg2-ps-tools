# dam_mpeg2_ps_utility

## Summary

DAM Karaoke machine needs a GOP index header in MPEG2-PS bitstreams. This software reads and writes correspondingly.

Also, DAM Karaoke machine needs End of sequence (EOS) and End of stream (EOB) NAL units in H.264 bitstreams. Please use it for encoding: https://github.com/DKKaraoke/ffmpeg-x264-add-eos-eob-patched

## List of verified DAM Karaoke machine

- DAM-XG5000[G,R] (LIVE DAM [(GOLD EDITION|RED TUNE)])
- DAM-XG7000[Ⅱ] (LIVE DAM STADIUM [STAGE])
- DAM-XG8000[R] (LIVE DAM Ai[R])
